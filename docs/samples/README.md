- AI 파싱 정확도 테스트
- UI/UX 검증
- 버그 재현 및 디버깅
- 데모 및 프레젠테이션

## 📁 파일 목록

현재 이 폴더에 샘플 PDF를 추가하면 자동으로 목록이 표시됩니다.

---

## ⚠️ 개인정보 보호

**중요**: 실제 고객 정보가 포함된 PDF는 절대 업로드하지 마세요!

### 안전한 샘플 준비 방법

1. **테스트 데이터 사용**
   - 가상의 고객명 (예: 홍길동, 김철수)
   - 임의의 날짜 및 금액

2. **개인정보 제거**
   - PDF 편집 도구로 실명 삭제
   - 주민번호, 주소 등 민감정보 제거

3. **익명화**
   - 고객명 → "고객 A", "고객 B"
   - 실제 데이터 → 더미 데이터

---

## 💾 파일 저장 규칙

### Git에 포함하지 않음 (기본)
- 대용량 PDF (> 1MB)
- 실제 고객 데이터 포함 가능성

### Git에 포함 가능
- 작은 샘플 PDF (< 1MB)
- 완전히 익명화된 테스트 데이터
- 스크린샷 이미지 (JPG, PNG)

---

## 📥 대용량 파일 공유 방법

대용량 PDF는 Git 대신 다음 방법 사용:

### 옵션 1: Google Drive
```
1. Google Drive에 업로드
2. 공유 링크 생성
3. 아래 목록에 링크 추가
```

### 옵션 2: Cloudflare R2 (프로젝트 전용)
```
1. Cloudflare R2에 업로드
2. 퍼블릭 URL 생성
3. 아래 목록에 링크 추가
```
